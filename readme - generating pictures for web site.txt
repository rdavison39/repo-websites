The program (WAG) used to generate the thumbnails and navigiation for the pictures of All, common, basement etc is called web album generator.  It stores projects as '.alm'.   Each directory (all, commone, basement, etc) represent an album and need to be regenerated if you edit, change, delete a picture.  The album project information (myAlumb.alb) resides in each directory under webs folder. For example  C:\\Rons Documents\Rons Personal Stuff\Webs\www.thedavisons.ca\205_burnett\generated\all

The album only generates all, commom and basement. All other web pages are crafted mannually

NOTE: WAG (web album generator - http://www.ornj.net/webalbum/  - i've stored the setup file in C:\Rons Documents\Rons Personal Stuff\Webs (webalbum_setup.exe) in case you ever need to reinstall it.

NOTE: You can order photo order by clicking on the photo and using Alt-UP or Alt-down to move pictures

To add pictures:
1. put pictures in appropriate directory common, basement, main_floor etc) (e.g. C:\Users\rdavison\Documents\Rons Documents\Rons Personal Stuff\205_Burnett\pictures to generate web site\main_floor)

2. run the copy_folders_into_all.bat - this will copy files from main_floor, common, basement into the 'all' directory. This file should be in something like (C:\Users\rdavison\Documents\Rons Documents\Rons Personal Stuff\99_Miller\pictures to generate web site) - C:\Users\rdavison\Documents\Rons Documents\Rons Personal Stuff\99_Miller\pictures to generate web site)

3. launch Web Album Generator and open the .alb file in the appropriate directory (you will need to follow the same steps for the all directory as well) - eg. the .alb file for main floor of 205 burnet would be located in C:\Rons Documents\Rons Personal Stuff\Webs\www.thedavisons.ca\205_burnett\generated\main_floor - IT WILL POINT TO T HE PICTURES IN ==>

4. add the pictures manually you want from the 'pictures to generate web site folder. using alt-up or alt-down put the photo in the sequence where you want it generated 
5. click "Generate Web Album"

6. Ignore first screen where it asks for Album title and accept all defaults on the subsquent screens

7. Make sure output folder is something like C:\Rons Documents\Rons Personal Stuff\Webs\www.thedavisons.ca\205_burnett\generated\common\

8. Do this for each directory that is reuired (main_floor, common, basement)
8. click finish
